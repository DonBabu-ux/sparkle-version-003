## QUICK REFERENCE: What each role can do

### 👤 REGULAR USER (Logged in)
- Manage own profile and account
- Create posts, stories, polls
- Like, comment, share
- Follow/unfollow users
- Send/receive messages
- Create/join groups
- Buy/sell on marketplace
- Create events, report lost items
- Block users, report content

### 👑 GROUP ADMIN (Within their group only)
- All regular user actions
- Remove members from group
- Ban users from group
- Promote/demote admins
- Delete any post in group
- Edit group info
- Delete group

### 🛡️ PLATFORM ADMIN (Global)
- All regular user actions
- View all profiles (even private)
- Delete any content (posts, comments, stories)
- Suspend or delete any user
- View all reports and resolve them
- View platform-wide analytics
- Send site-wide announcements
- View activity logs
- Manage featured content
